Distribuições Linux

Uma Distribuição Linux (geralmente abreviada como Distro) é um sistema operativo criado a partir de uma coleção de software com base no kernel do Linux e, geralmente, um gestor de pacotes e repositório de programas.

Em baixo encontrão-se algums exemplos de distribuições e as suas características

Slackware é uma das distribuições mais antigas do Linux e a mais antiga que ainda recebe atualizações nos dias de hoje. Algumas das suas principais características são:

  • Estabilidade através da simplicidade
  • A destribuição Linux mais similar ao Unix
  • Ausência de um instalador e inteface gráfica

Devido à sua simplicidade e funcionalidades conservativas, o Slackware é geralmente considerado uma distribuição Linux destinada a usuários já com um vasto conhecimento sobre o Linux. Algumas outras distribuições destinadas a esses usuários são por exemplo o Arch Linux, Gentoo Linux e Linux From Scratch.

Debian é uma distribuição Linux especialmente conhecida pelo seu gestor de pacotes, chamado APT, que permite facilmente a instalação, atualização e desinstalação de programas. Algumas das suas principais características são:

  • Uso do gestor de pacotes APT
  • Facilidade de uso através de interfaces gráficas
  • Base de diversas outras distribuições Linux

Lançado em 1993, o Debian é uma das distribuições Linux mais antigas ainda em desenvolvimento, possui versões para mais de 20 arquiteturas de processadores e é a distribuição Linux em que a maior quantidade de outras distribuições tomam base.

Red Hat Linux (por vezes abreviado como RHL) é uma distribuição Linux criada pela Red Hat para ambientes comerciais e servidores conhecida pela sua estabilidade e fácil manutenção. Algumas das suas principais características são:

  • Estabilidade e fiabilidade em ambientes empresariais
  • Fácil instalação, uso e manutenção
  • Uso do gestor de pacotes RPM

Seguindo-se a sua descontinuação em 2004, o Red Hat Linux foi separado em dois sucessores, o Fedora Linux para uso pelo consumidor e o Red Hat Enterprise Linux para ambientes empresariais.
